# **App Name**: SmartTasker

## Core Features:

- Panel de Control Dinámico: Visualización centralizada del estado académico, tareas pendientes, eventos futuros y métricas de progreso del estudiante.
- Gestión de Eventos y Horarios: Permite a los usuarios crear, editar y eliminar eventos en el calendario, especificando categorías, fechas, horas y descripciones. Los eventos se registran en una base de datos.
- Navegación Intuitiva: Un riel de navegación lateral delgado y basado en iconos para acceso rápido a las secciones clave de la aplicación como Dashboard (Home), Calendario Completo, Checklist de Tareas y Configuración.
- Barra de Cabecera Global: Incorpora un campo de búsqueda global, un icono de campana de notificaciones con badge de alerta, y un avatar de perfil de usuario para acceso a opciones de cuenta y ajustes.
- Seguimiento de Progreso Estudiantil: Muestra tarjetas interactivas con gráficas circulares de rendimiento académico, brindando una visión del progreso semanal y mensual del estudiante.
- Acciones Rápidas Contextuales: Un panel de botones tipo 'Pill' para acceder rápidamente a funcionalidades comunes como 'Acceder a Clases', 'Entregar Tareas' y 'Ver Calificaciones'.
- Recomendaciones Inteligentes de Estudio: Una herramienta de inteligencia artificial que ofrece sugerencias personalizadas de bloques de estudio y enfoques, analizando las tareas pendientes y el historial de rendimiento del usuario para optimizar la planificación académica.

## Style Guidelines:

- El esquema de color general es claro. El color de fondo principal es un gris muy tenue (#F8F9FD), que proporciona un lienzo limpio y profesional para la interfaz de usuario.
- El color primario para acciones y estados activos es un Morado Eléctrico vibrante (#6D4AFF). Este color resalta elementos interactivos y proporciona un acento energético en contraste con el fondo claro.
- Un color de acento complementario es un azul brillante (#2E4AF4), usado para añadir diversidad visual y destacar ciertos elementos analógicos al tono principal morado, manteniendo una armonía vibrante.
- Se utilizarán colores pastel (azul, rosa, verde pálido) de la paleta de soporte para diferenciar y categorizar las tarjetas de contenido, siguiendo el sistema de diseño SoftFlow UI para una estética suave y organizada.
- La tipografía principal será 'Inter' (sans-serif), que aporta un toque moderno y legible a la interfaz. Se aplicará una jerarquía clara con títulos en negrita y cuerpo de texto en tonos de gris oscuro para optimizar la legibilidad.
- Se utilizarán iconos minimalistas y funcionales a lo largo de la aplicación, especialmente en la navegación lateral y las notificaciones, para complementar el diseño 'Soft-UI' y asegurar una experiencia de usuario intuitiva.
- El diseño de la aplicación será de una sola página, estructurado con una navegación lateral izquierda delgada, una barra de encabezado global y una cuadrícula principal de tarjetas en el contenido. Las tarjetas presentarán bordes muy redondeados (24px a 32px) y sombras suaves (box-shadow) para un efecto de profundidad.
- Se implementarán animaciones sutiles, como un ligero escalado y la intensificación de sombras en los estados al pasar el cursor (hover), con una transición de 0.3 segundos, para ofrecer una experiencia interactiva y fluida.
